# brain-quoridor
## 概要
コリドールをWebで遊べるようにしたもの。AIあるいはオンライン対戦ができます。
オリジナル要素としてアイテム機能を追加予定。

## 起動方法
### 本番環境版
1. このリポジトリを持ってくる
```bash
git clone https://github.com/tochiman/brain-quoridor.git
```
2. コンテナの起動(本番環境版)
```bash
docker compose up -d -f compose-prod.yaml
```
### 開発環境版
1. このリポジトリを持ってくる
```bash
git clone https://github.com/tochiman/brain-quoridor.git
```
2. コンテナの起動(本番環境版)
```bash
docker compose up -d
```
